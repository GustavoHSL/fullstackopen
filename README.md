# fullstackopen

Este repositório contém meus exercícios e projetos desenvolvidos durante o curso **Full Stack Open**, oferecido pela Universidade de Helsinki.

## 📚 Sobre o curso

O Full Stack Open aborda o desenvolvimento moderno de aplicações web com JavaScript, incluindo:

- React
- Node.js
- Express
- MongoDB
- GraphQL
- TypeScript
- Testes
- CI/CD

## 🗂 Estrutura do repositório

- `part0` – Fundamentos da web e diagramas
- `part1` – Introdução ao React
- `part2` – Comunicação com servidor
- `part3` – Backend com Node.js e Express
- `part4` – Testes e autenticação
- `part5` – Testes em frontend
- `part6` – Gerenciamento de estado
- `part7` – React Router, custom hooks e estilos
